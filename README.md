# docs
## 程序功能描述  
1.用户可以看到推荐商品  
2.用户可以查找想要购买的偶像的商品  
3.用户可以将想要购买的商品加入购物车  
4.用户可以查看购物车  
5.用户可以查看商品的评价  
6.用户可以查看商品的详细信息  
7.用户可以查看自己喜欢的偶像的商品列表  
8.用户可以对已经购买的商品进行评价
